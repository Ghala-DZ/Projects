# Code & Crumble

A user-friendly web application for sharing and managing dessert recipes.

## Project Type
Academic group project developed as part of a Web Development course.

## Features
- User registration and authentication
- Add, edit, and browse dessert recipes
- Recipe of the Day functionality
- Responsive and easy-to-use interface

## Technologies Used
- Frontend: HTML, CSS, JavaScript, React
- Backend: PHP
- Database: MySQL

## Notes
This project was developed collaboratively. Team member details were omitted for privacy.
