#  Safe Haven Watch

Safe Haven Watch is a smart safety system designed to help locate lost children in crowded public places and quickly reunite them with their guardians.

---

##  Project Type

Academic group project developed as part of a Software Engineering / Systems Analysis course.

---

## Features

* User registration and secure authentication
* Real-time GPS tracking of children
* Safe zone (geofencing) monitoring with instant alerts
* SOS emergency alerts with live location
* Voice messaging and calling between guardian and child
* Lost child incident reporting and tracking
* Battery status and device tampering notifications
* Secure data storage and privacy protection

---

##  Technologies Used

* Mobile Application: GPS & map-based interface
* Communication: Secure notifications, VoIP, SMS
* Backend: Server-side processing and data management
* Database: Secure storage for user and incident data
* Embedded System: GPS-enabled wearable device
