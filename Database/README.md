## Cinema Management System

An academic database application designed to manage and organize cinema-related data including films, directors, actors, genres, and ratings.

## Description
The Cinema Management System is a database-driven application that models real-world cinema operations.  
It focuses on organizing film information and managing relationships between films, directors, actors, genres, and ratings using structured database design and SQL implementation.

## Key Features
- Manage films with unique IDs, titles, languages, release dates, and genres
- Associate each film with exactly one director
- Manage actors and their roles in different films
- Store multiple ratings for each film
- Organize films by genres
- Maintain data integrity through relational constraints

## Program Flow
- Define cinema business rules
- Design ER models (UML & Chen)
- Convert ER models to relational schema
- Create database tables using SQL
- Insert sample data into tables
- Execute SQL queries and view results

## Technologies & Concepts
- SQL
- Relational Databases
- ER Modeling (UML Diagram & Chen Model)
- Relational Schema Design
- Data Integrity and Constraints
