# Abaya Shopping Cart System

An online shopping application that allows users to customize and purchase abayas based on their preferences.

## Project Type
Academic group project developed as part of a university coursework.

## Description
The Abaya Shopping Application is a customizable e-commerce system where users can design abayas by selecting fabric, style, color, and size, with real-time price updates. The system provides a smooth shopping cart experience from customization to checkout.

## Key Features
- Customizable abaya products (fabric, style, color, size)
- Real-time price calculation based on selections
- Shopping cart management (add, edit, remove items)
- Secure checkout and credit card payment processing
- Order summary and confirmation after purchase

## Program Flow
- User information setup
- Abaya customization process
- Review and modify selected items
- Shopping cart management
- Checkout and payment
- Order confirmation

## Technologies & Concepts
- Object-Oriented Programming
- User input validation and exception handling
- UML diagrams (Class Diagram)
- Modular and user-centric program design
